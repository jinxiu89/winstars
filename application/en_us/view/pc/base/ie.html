Browser upgrade

If you are using an earlier version of Internet Explorer (IE11 or the browser using the kernel), you won't be able to access the site until you upgrade your browser.

Because Microsoft has no longer provided support and updates for Internet explorer 11 since January 12, 2016. That is, without browser itself security updates, your PC can be vulnerable to attack by harmful viruses, spyware, malware that will steal or damage your important data and information. Further information, please refer to Microsoft's support for earlier versions of Internet Explorer. Note: Support for Windows XP and earlier versions of Internet Explorer are ceased.
